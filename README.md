# Placement-Tracker-Project
This is a simple placement tracker fullstack project made using FastAPI (Python template) as the backbone component.

## Features
- Add placement applications
- View all applications
- Automatic selection/rejection based on CGPA (>= 6.5)
- Dashboard with:
  - Total Applications
  - Selected
  - Rejected
- SQLite database integration

## Technologies
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript

## Run

Backend

uvicorn main:app --reload

Frontend

Open PlacementTracker.html using Live Server.
